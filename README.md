# Digital-Communications-
# Sampling Theorem and Signal Reconstruction

## Overview
This repository contains MATLAB simulations and a LaTeX report for
Experiment 1 (Sampling Theorem Analysis) and Experiment 2 (Signal Reconstruction).

## Code Structure
- `matlab/provided/` : Base MATLAB code provided by the lecturer
- `matlab/modified/` : Modified and extended MATLAB scripts
- `figures/` : Output plots used in the report
- `report/` : LaTeX source files

## How to Run
1. Open MATLAB
2. Run `ex1_sampling_analysis.m`
3. Run `ex2l1_reconstruction.m`

## Notes
Sampling frequency used: 50 Hz  
Message signal components: 1 Hz and 3 Hz
